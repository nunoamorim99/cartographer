# Bundled fonts

These fonts are bundled so Cartographer's PDFs render identically on every machine,
independent of what fonts the host has installed.

- **Spectral** (body) — © Production Type. SIL Open Font License 1.1. See OFL-spectral.txt
- **Fira Sans** (headings) — © The Mozilla Foundation & Telefónica. SIL Open Font License 1.1.
  See OFL-firasans.txt

Both licenses permit bundling and redistribution. DejaVu (a system font) is kept only as a
fallback in style.css if these ever fail to load.

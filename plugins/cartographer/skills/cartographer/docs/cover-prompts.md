# Cartographer — cover art prompts (Higgsfield)

Ready-to-run prompts for generating cover art in a Higgsfield-connected environment
(e.g. Claude Code with the Higgsfield MCP). Model: **`nano_banana_pro`** — it renders
in-image text ("CARTOGRAPHER") most reliably. Aspect ratio: **wide banner (16:9 or ~3:1)**
for a README header. Keep the shared **style prefix** on every prompt for a consistent set.

The bundled designed covers (`docs/assets/cover.png`, `cover-alt.png`) are B&W and match the
skill's own typography; these prompts are for evocative marketing variants.

---

## Shared style prefix (prepend to every prompt)

> Sophisticated editorial cover art for a developer tool, refined and minimal, high-contrast
> black and white with subtle warm paper texture, fine ink linework, elegant serif display
> typography, generous negative space, professional and understated — no clutter, no gradients,
> no photographic realism.

## Variant 1 — Antique map fusion

> …an antique cartographic chart whose coastlines, contour lines and rivers are subtly formed
> from interconnected repository nodes and thin circuit traces; a pair of brass dividers and a
> small compass rose rest on the chart; the title **"CARTOGRAPHER"** in engraved serif capitals
> across the upper third, with a small line of smaller text reading "a Claude Code skill".

## Variant 2 — Network becomes terrain

> …a constellation of small circular nodes connected by fine straight lines that gradually
> resolve into the nested contour lines of a topographic map, as if a codebase were turning into
> mapped territory; clean, modern, mostly white space; the word **"CARTOGRAPHER"** set large in
> a confident serif on the left, a thin horizontal rule beneath it.

## Variant 3 — Cartographer's desk (flat lay)

> …an overhead flat-lay of a cartographer's desk: an unrolled map, a brass compass, a
> magnifying glass and an ink pen, with faint terminal windows, folder icons and file trees
> woven softly into the map's surface; the title **"CARTOGRAPHER"** hand-lettered in an elegant
> serif along the top edge.

---

## Tips

- Generate 2–4 per variant and pick; text rendering varies run to run — regenerate if
  "CARTOGRAPHER" is misspelled rather than editing it.
- Keep the same style prefix across a batch so the set looks like a family.
- If you want the cover to match the documents exactly, prefer the bundled B&W designed cover;
  use these AI variants for social/marketing where a richer image helps.

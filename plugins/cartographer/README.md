# Cartographer (plugin)

Generate stakeholder documentation from a project's repositories — what each product does and how it fits together, as black-and-white Markdown + PDF. Read-only, audience-aware, provenance-tracked.

Bundles the **skill** (`skills/cartographer/`) and three **commands** (`commands/`): `/assess`, `/generate`, `/doc-status` (namespaced as `/cartographer:…` once installed).

Full documentation is in the skill's own README: [`skills/cartographer/README.md`](skills/cartographer/README.md). See the repository root README for installation.
